BMW M5 Landing Page
A high-performance, mobile-responsive landing page dedicated to the "Monster" BMW M5. This project focuses on sleek UI design and smooth user interactions using modern web technologies.

🚀 Features
Fully Responsive Design: Optimized for all screen sizes using Tailwind CSS.

Interactive Navigation: Mobile-friendly navigation drawer with a sliding effect and backdrop overlay.

Performance Stats: Detailed breakdown of the BMW M5's technical specifications.

Modern UI: Clean, dark-themed aesthetic matching the BMW M-series branding.

🛠️ Tech Stack
HTML5: Semantic structure.

Tailwind CSS: Utility-first styling for responsiveness and custom layouts.

JavaScript (ES6): Logic for the mobile menu toggle and UI interactions.
